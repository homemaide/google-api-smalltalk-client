# google-api-smalltalk-client

## Supported Smalltalk Versions
Pharo Smalltalk 4.0, 5.0

## Installation
```smalltalk
Metacello new
    baseline: 'GoogleAPI';
    repository: 'github://newapplesho/google-api-smalltalk-client:v0.1/pharo-repository';
    load.
```
